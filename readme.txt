WP Travel Custom Payment

== Description ==
"WP Travel Custom Payment" is the addons plugin for WP Travel. Which made for Custom Payment payment process. WP Travel Custom Payment requires WP Travel version 1.2.3 or above.

= Release Notes: =
Before updating plugin please check ["our latest release notes"](http://wptravel.io/category/wp-travel-release/).

* Custom Payment system

== Installation ==
= Uploading in WordPress Dashboard =
* Navigate to the 'Add New' in the plugins dashboard
* Navigate to the 'Upload' area
* Select wp-travel-custom-payment.zip from your computer
* Click 'Install Now'
* Activate the plugin in the Plugin dashboard

= Using FTP =
* Download wp-travel-custom-payment.zip
* Extract the wp-travel directory to your computer
* Upload the wp-travel directory to the /wp-content/plugins/ directory
* Activate the plugin in the Plugin dashboard
